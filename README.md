VOX Container
=============

Features
--------

* Various environments: xhyve/lxc/runc/openvz/iocage
* Core Linux with Synrc Erlang and devtools

Install on Mac
--------------

```
brew install squashfs on mac
npm i nugget -g
./download.sh
./build.sh
./xhyve.sh
````
